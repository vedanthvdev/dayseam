# v0.1 Phase 1: Foundations — Implementation Plan

> **For agentic workers:** this plan is executed PR-by-PR. Each task below is one PR on its own branch. Follow the one-commit-per-branch rule: the first change on a branch is a real `git commit`; every subsequent change is `git commit --amend --no-edit` followed by `git push --force-with-lease`. When a step below says "Commit," obey that rule.

**Goal:** Build the development spine so every later phase just drops code into already-wired crates. At the end of Phase 1, `pnpm tauri dev` launches a Mac window with a themed, empty layout; `cargo test --workspace` and `pnpm -r test` both pass; CI is green on every PR; branch protection forbids direct commits to `master`.

**Architecture:** Tauri desktop app (Rust core + React/TS frontend) in a dual-workspace monorepo (Cargo + pnpm). All IO lives in Rust async crates; the frontend only talks through typed Tauri IPC. No user-visible behaviour yet — this phase establishes interfaces, persistence, secret storage, event plumbing, and the app shell.

**Tech Stack:** Rust stable 1.77, Tokio, `sqlx` (SQLite), `keyring` (macOS Keychain), `reqwest`, `ts-rs`, `thiserror`, `insta` (snapshots), `proptest` (property-based), React 18, TypeScript 5, Tailwind 3, Vitest, MSW.

              }

- [ ] **Step 2.6: Write the error enum with stable codes**

  `crates/dayseam-core/src/error.rs`:
  ```rust
  use thiserror::Error;

  #[derive(Debug, Error, Clone, serde::Serialize, serde::Deserialize, ts_rs::TS)]
  #[ts(export, export_to = "../../packages/ipc-types/src/generated/")]
  #[serde(tag = "variant", content = "data")]
  pub enum DayseamError {
      #[error("auth error [{code}]: {message}")]
      Auth { code: String, message: String, retryable: bool, action_hint: Option<String> },
      #[error("network error [{code}]: {message}")]
      Network { code: String, message: String },
      #[error("rate-limited [{code}]: retry after {retry_after_secs}s")]
      RateLimited { code: String, retry_after_secs: u64 },
      #[error("upstream changed [{code}]: {message}")]
      UpstreamChanged { code: String, message: String },
      #[error("invalid config [{code}]: {message}")]
      InvalidConfig { code: String, message: String },
      #[error("io error [{code}]: {message}{}", path.as_ref().map(|p| format!(" at {}", p.display())).unwrap_or_default())]
      Io { code: String, path: Option<std::path::PathBuf>, message: String },
      #[error("internal error [{code}]: {message}")]
      Internal { code: String, message: String },
  }

  impl DayseamError {
      pub fn code(&self) -> &str {
          match self {
              Self::Auth { code, .. }
              | Self::Network { code, .. }
              | Self::RateLimited { code, .. }
              | Self::UpstreamChanged { code, .. }
              | Self::InvalidConfig { code, .. }
              | Self::Io { code, .. }
              | Self::Internal { code, .. } => code,
          }
      }
  }
  ```
